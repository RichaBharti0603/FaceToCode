# 🌌 FaceToCode Pro
### *High-Performance Neural ASCII Visualization & Global Discovery Mesh*

![Version](https://img.shields.io/badge/version-2.5.0-green.svg?style=for-the-badge)

**FaceToCode Pro** is a cinematic, real-time ASCII rendering system that transcends traditional filters. It combines a custom high-performance rendering pipeline with Google's Gemini 1.5 Flash to create a dynamic, emotion-reactive cyberpunk visualization engine. 

Explore the **Discovery Mesh**, remix community creations, and broadcast your neural fragments to a global audience.

---

## 📽️ The Core Engine

### 🧠 Neural Emotion Sync
Leveraging the power of **Gemini 1.5 Flash**, the engine continuously monitors facial expressions to dynamically adapt its visual DNA:
- **😊 Happy → Color Mode**: Vibrant, high-fidelity RGB mapping.
- **😐 Neutral → B&W Mode**: Classic high-contrast terminal aesthetic.
- **😠 Serious → Matrix Mode**: Iconic cascading rain with temporal per-column tracking.

### 🌐 The Discovery Mesh (`/explore`)
A global, decentralized feed of neural snapshots. 
- **Broadcast**: Share your captures to the public community wall.
- **Remix DNA**: Found a style you love? One-click "Remix" allows you to clone the exact engine settings (density, pixel mapping, contrast) of any public snapshot.

### 🎨 Custom Visualization Models
- **Matrix Rain**: Custom per-column state machines for authentic vertical cascading effects.
- **Amber Glow**: Precision-mapped phosphor effects for vintage hardware simulation.
- **Blocks & Binary**: Specialized character sets for abstract high-density data representation.

## 🚀 Pro-Grade Features

## 🚦 System Requirements & Setup

### Prerequisites
- Node.js 20.x or higher
- A valid **Google AI Studio API Key**
- A **Supabase** project (Storage bucket + PostgreSQL)

### Rapid Deployment

1. **Clone the Mesh**:
   ```bash
   git clone https://github.com/your-username/FaceToCode.git
   cd FaceToCode
   ```

2. **Initialize Environment**:
   Create a `.env` file:
   ```env
   VITE_GEMINI_API_KEY=your_key
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key
   VITE_POSTHOG_KEY=your_key
   ```

3. **Install Dependencies**:
   ```bash
   npm install
   ```

4. **Boot System**:
   ```bash
   npm run dev
   ```

## 🛡️ Privacy & Neural Ethics

FaceToCode Pro is designed with a **privacy-first** philosophy:
- **Local Processing**: Core ASCII conversion and frame processing occur entirely on the client's GPU/CPU.
- **Volatile Neural Data**: Frames sent to Gemini for emotion detection are heavily downsampled and never persisted on our servers.
- **Local Identity**: User profiles and analytics data remain exclusively in your browser's private storage unless you explicitly "Broadcast" a snapshot.
